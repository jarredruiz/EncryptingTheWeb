Secure Socket Layer

Secure Socket Layer (SSL) is a security protocol thats main purpose is to provide three things mentioned in HTTPS: Confidentiality, Integrity, and Authenticity. Before HTTPS and SSL, all transactions between a user and website was not encrypted. Meaning that any information of the user was sent over the web in plaintext which can easily be obtained by hackers. SSL works by using a handshake encryption method. The computer first makes a request to the server, then the server then will send a public key to the user who will then send an encrypted one back. If the keys match for both server and client, then the connection can be established and transmit secure data.
